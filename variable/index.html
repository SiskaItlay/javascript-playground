Variabel JavaScript

Variabel dalam JavaScript digunakan untuk menyimpan nilai data.

Variabel dideklarasikan menggunakan kata kunci const untuk mendeklarasikan variabel yang nilainya tetap (konstan) dan tidak dapat diubah, let untuk mendeklarasikan variabel yang nilainya dapat diubah, atau var untuk mendeklarasikan variabel dengan cakupan (scope) yang lebih luas.
